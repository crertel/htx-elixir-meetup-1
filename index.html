# Houston Elixir 2016-09-07 ## Installing Elixir
## Windows 1. Download the package from https://repo.hex.pm/elixir-websetup.exe 2. Run it.
## Mac (Homebrew) ``` $ brew update $ brew install elixir ```
## Mac (Macports) ``` $ sudo port install elixir ```
## 'Nix (Arch) ``` $ pacman -S elixir ```
## 'Nix (openSUSE) ``` $ zypper ar -f http://download.opensuse.org/repositories/devel:/languages:/erlang/openSUSE_Factory/ erlang $ zypper in elixir ```
## 'Nix (Gentoo) ``` $ emerge --ask dev-lang/elixir ```
## 'Nix (GNU Guix) ``` $ guix package -i elixir ```
## 'Nix (Fedora/RHEL/CentOS) ``` $ yum update $ yum install elixir ```
## 'Nix (FreeBSD) Ports: ``` $ cd /usr/ports/lang/elixir && make install clean ``` Package: ``` $ pkg install elixir ```
## 'Nix (Ubuntu/Debian) ``` $ wget https://packages.erlang-solutions.com/erlang-solutions_1.0_all.deb && sudo dpkg -i erlang-solutions_1.0_all.deb $ sudo apt-get update $ sudo apt-get install esl-erlang $ sudo apt-get install elixir ```
